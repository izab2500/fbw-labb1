Laboration 1

Syftet är att förstå hur NodeJS kan underlätta utveckling av webbplats, automatisera uppgifter och skapa en utvecklingsmiljö med Vite, Git, GitHub och Netlify för automatisk publicering.

Git och GitHub

Använd Git och GitHub för versionhantering. Gör regelbundna commits med tydliga meddelanden och ha en .gitignore-fil för att inte skicka med onödigt stora filer som man kan återskapa med npm install. Arbeta med branches och gör repot publikt.

Automatisering och Publicering

Skapa en struktur för webbutveckling med HTML, CSS, JavaScript och bilder. Använd NodeJS och byggverktyget Vite för automatisering, minifiering av CSS/JavaScript och bildoptimering.

Skapa Demowebbplats

Utveckla en responsiv webbplats med minst tre sidor: en startsida för att beskriva uppgiften, en bildsida med minst fem bilder och en sida för att beskriva arbetsprocessen.

Publicering

Publicera webbplatsen på en host som inte använder FTP/SFTP, som exemplevis Netlify. Använd branches för utveckling och publicering, och inkludera en länk till den publicerade webbplatsen i README-filen.
